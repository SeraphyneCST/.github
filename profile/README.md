# Seraphyne Computer Science Group

Seraphyne is an independent open-source research and development community with interests in:  
- **Large Language Models (LLMs)** - training, optimization, and deployment of advanced natural language processing systems.  
- **Android Engineering** - frameworks, and native development.  
- **Malware Engineering & Security Research** - reverse engineering, threat analysis, and security tool development.  
- **Web Development** - modern frontend frameworks, scalable backend systems, and component-based architecture for robust applications.  

---

> **Seraphyne** is a collective research circle:  
> open, collaborative, and driven by curiosity in applied STEM fields.
